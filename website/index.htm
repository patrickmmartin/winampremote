What is the winamp remote control?

The short answer: a client-server "suite" for controlling a single winamp instance from multiple client machines over the network.

The program documentation should answer most questions.

The program license details the stringent terms of use ;-).

Latest enhancements:

  • Release of the source under the GPL
  • Enhancement to server API in 1.3.8!
    Due to changes to network interface to support the FreeDCE project I cannot warrant the new binaries are backwards compatible at all. However, as a result for these changes, there is the prospect of linux client applications and all that brings with it.
  • Changed packages
    Now there are separate packages for the client and the plugin and the source. These are smaller and should work out of the box for anyone having the MSI installer. If not, the MSI installer can be obtained from Microsoft.
