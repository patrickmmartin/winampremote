Detailed Description

Class to wrap up handling the RPC runtime call-backs. The RPC functions are linked in statically as a link demand from the generated stubs. The handler for the winamp methods implements generic behaviour, and for different implementations the correct handler needs to be invoked for the mocked winamp or the actual winamp process. The endpoint and protocol sequence are kept as the API strings and are default initialised.
